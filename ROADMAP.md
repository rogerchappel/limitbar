# Roadmap

## V1: Local CLI core

- Local snapshot adapters.
- Threshold warnings.
- Text and JSON output.
- Fixtures and smoke tests.

## V1.1: Better operator ergonomics

- `limitbar init` config generator.
- JSON schema for config validation.
- Compact one-line renderer.
- More example fixtures.

## V2: Menu-bar shell

- Native macOS wrapper that polls `limitbar status --format json`.
- Popover with provider details and warnings.
- Explicit local notification settings.

## Later

- Optional safe network adapters where providers offer documented APIs.
- Orchestrator queue-depth integrations.
- Exportable status history for local trend analysis.
